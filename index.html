Algolia Places

Free autocomplete address search

Why Places?

Algolia Places is our answer to provide a fast, distributed and easy to use address search autocomplete library for your website.

It harness the impressive OpenStreetMap’s open source database of worldwide places.

Get Started

Easy to use

To use Algolia Places, you only need to add an HTML <input /> and the places.js library. It takes 1 minute to install it on any website.

Relevance

We have been building search engines for years. It felt natural to build an address search using Algolia's relevance knowledge.

Typo tolerance

Algolia Places supports typos and mistakes like any Algolia search index. We have a unique way to handle typos and rank results accordingly.

Speed

Algolia Places deliver address search in milliseconds to your users. No matter where they are: we automatically route them to our closest server.

Free to use

Algolia Places is free. There’s a rate limit of 1000 searches per IP per hour so you should be safe as long as your users are not doing more than 1000 searches per hour.
